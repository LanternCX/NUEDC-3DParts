# NUEDC-3DParts

为 2025 电赛画的一些结构件

具体说明请转移每一个文件夹的 `Readme.md`

## 目录结构

```bash
├─5mW Laser Pen Casing # 5mW 激光笔外壳
├─Adapter Board # 云台到ESP32开发板的转接板
├─Base # 两种摄像头支架底座
├─Camera Base # MaixCam-Pro 摄像头底座
├─Fixings # 固定摄像头到 80mm 碳纤维管的固定件
├─Frange # 两种 3D 打印法兰轴
├─Magnetic Base # 可拆卸元素的磁吸底座
└─PTZ # 步进电机云台
```
